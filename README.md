# Debian Jessie Git Install

This bash script demonstrates the standard procedure for installing Git on a Debian 8 (Jessie) system using apt-get. It updates package lists and then installs the git package, followed by a verification step.

## Language

`bash`

## How to Run

Save the code as install_git.sh, make it executable with 'chmod +x install_git.sh', and run it with './install_git.sh'.
